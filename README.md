# Philosophers
### _Симуляция задачи об обедающих философах_
- *Для компиляции // бонусы*
```sh
make // make bonus
```
- *Запускать*
```sh
./philo количество_философов время_жизни время_есть время_спать
```
- *Те же самые аргументы с бонусами*
```sh
./philo_bonus 
```
